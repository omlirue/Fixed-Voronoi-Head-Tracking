This is a novel, discrete head-cursor system to help people with motor disabilities interact with technology.
1. This system uses Voronoi geometry to designate regions to interactive targets on a screen.
2. Targets are activated by entering Voronoi regions and dwelling for a while.
3. This system is discrete, so there is no visible mouse or overt precision required for navigation.
4. We evaluate our system with a modified version of Fitts Law optimized for this architecture.
