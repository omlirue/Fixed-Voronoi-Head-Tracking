One Euro / Exponential Filter Equivalence Verification

Professor's Check: When velocity = 0, One Euro with fc_min should equal Exponential with equivalent alpha

Mathematical Basis (from One Euro Filter paper):

Cutoff frequency: fc = fc_min + beta * |derivative|
When velocity = 0: derivative = 0, so fc = fc_min

Time constant: tau = 1 / (2 * pi * fc)
Smoothing factor: alpha = 1 / (1 + tau / Te)
Where Te = 1/samplingRate = 1/60 s (sampling period)

Therefore when stationary:
alpha = 1 / (1 + 1/(2*pi * fc_min * Te))
alpha = 2*pi * fc_min * Te / (1 + 2*pi * fc_min * Te)

Inverse (alpha -> fc_min):
fc_min = alpha / (2*pi * Te * (1 - alpha))
