Note on the low r² values. In this study, each participant's MT samples come from three very different filter operating points (Low / Med / High latency × 2 filter types). 7. Power analysis & recommended n for the real study

Per-participant ΔTP (One Euro − Exponential): mean = -0.009 bits/s, SD = 0.208, Cohen's dz = -0.043 across 12 paired observations.

Pilot effect size for ΔTP is essentially zero (d_z = -0.043), so a paired-t test cannot estimate a meaningful target n. With n=12 (HCI standard) a real study could detect d_z ≥ 0.89; with n=20 it could detect d_z ≥ —. Pilot does not yet support a strong directional claim on filter type.

Reporting minimum detectable effect (MDES) at standard sample sizes instead of a target n:

  • n = 12 → minimum detectable dz ≈ 0.89
  • n = 20 → minimum detectable dz ≈ —

Practical reading: a real study with n=12 would only catch a medium filter effect; n=20 catches a small-to-medium effect. If you expect the true effect to be small (which the pilot suggests), plan for n ≥ 20.

Paired-t power on per-participant ΔTP collapsed across variance pairs. Conservative estimate. For final study, refine with simr in R or a simulation-based mixed-model power analysis.
