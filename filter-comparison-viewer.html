🔬 Filter Comparison Viewer

Compare One Euro Filter vs Exponential Smoothing Performance

How to Use:

ℹ️ Sample data loads automatically! You'll see demo graphs below. To use your own data, simply click the upload boxes to select your CSV files.

  1. Export both Pareto front CSVs from your parameter optimization system
  2. Upload the One Euro Filter CSV in the left box
  3. Upload the Exponential Smoothing CSV in the right box
  4. View three visualizations:
    • 📊 Individual Filter Graphs - See each filter's Pareto front separately with its own scale
    • ⚔️ Direct Comparison - Both filters overlaid on the same scale to compare performance
  5. Tip: Double-click either upload box to reload sample data
  6. Hover over points in any graph to see detailed parameter values
